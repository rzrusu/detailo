# Detailo

A microservices-based CMS platform for auto-detailing shop owners. Manage your business, services, staff, and let customers book appointments through a public shop page.

## Tech Stack

**Backend:** Java 25, Spring Boot, Spring Cloud, Spring Security (OAuth2/JWT), Hibernate/JPA, PostgreSQL, Redis, Kafka, Flyway

**Frontend:** React 19, TypeScript, Vite, Tailwind CSS, TanStack Query, Zustand

**Infrastructure:** Docker, Kubernetes, Helm, Terraform, GitHub Actions

**Observability:** OpenTelemetry, Prometheus, Grafana, Loki, Tempo
